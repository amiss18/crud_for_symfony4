# Symfony 4 : Getting Started

## clone the project


## Run server

`$ php bin/console server:run`

 and open your browser and access the **http://localhost:8000**
 
 
 

## Execute this command to run tests:

`
$ ./vendor/bin/simple-phpunit
`
